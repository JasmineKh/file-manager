## file manager Application

### Technologies Used
*  Spring Boot
*  Maven
*  Docker

###  IDE used
*  IntelliJ IDEA

###  Run project
Prerequisites:
* JDK17
* Maven
* Docker
* docker-compose
* postman

For up and running the application, just run the following command:
* `docker-compose up --build`


Rest endpoint can be called by _**file-manager.postman_collection.json**_
